# Joining the 5-Day AI Agents Intensive as part of Kaggle Growth Lab

I am adding a new track to my Kaggle Growth Lab.

From June 15 to 19, 2026, I will follow the 5-Day AI Agents: Intensive Vibe Coding Course with Google on Kaggle.

The course is interesting to me because it connects several things I want to practice in public:

- building agents beyond simple chatbots
- using tools and APIs inside agent workflows
- designing memory and context
- evaluating quality and security
- turning prototypes into deployable systems

I do not want this to be just another course I privately complete.

So I am treating it as a public engineering log.

My plan:

- Kaggle is where I complete the assignments and capstone.
- GitHub is where I keep notes, experiments, and implementation logs.
- LinkedIn is where I explain the lessons in public.

The capstone project, Kaggriculture, will be especially useful because it requires an autonomous agent to make decisions in a simulation environment.

That means the interesting part is not only whether the agent performs well.

The interesting part is how I design the agent, test it, improve it, and explain the tradeoffs.

# Kaggle Growth Lab에 AI Agents 트랙을 추가합니다

Kaggle Growth Lab에 새로운 트랙을 추가합니다.

2026년 6월 15일부터 19일까지 Kaggle에서 진행되는 Google의 5-Day AI Agents Intensive 과정을 따라갈 예정입니다.

이 과정이 흥미로운 이유는 단순히 AI agent를 써보는 것이 아니라, 제가 공개적으로 연습하고 싶은 주제들이 한 번에 연결되기 때문입니다.

- chatbot을 넘어서는 agent 구조
- tool과 API를 사용하는 workflow
- memory와 context 설계
- quality와 security 평가
- prototype을 실제 프로젝트로 정리하는 과정

이번에도 혼자 끝내는 강의 수강으로 남기지 않으려고 합니다.

Kaggle은 실습과 capstone을 진행하는 곳으로 사용합니다.
GitHub에는 노트, 실험, 구현 로그를 남깁니다.
LinkedIn에는 배운 점과 판단 과정을 공유합니다.

특히 capstone인 Kaggriculture는 simulation 환경에서 autonomous agent가 의사결정을 해야 하는 프로젝트라서 좋습니다.

중요한 것은 단순히 agent 성능이 좋았는지가 아닙니다.

어떤 기준으로 agent를 설계했고, 어떻게 테스트했고, 무엇을 개선했는지를 보이는 것이 이번 목표입니다.
