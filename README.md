# Kaggle Growth Lab

This repository documents my journey of growing through Kaggle competitions, notebooks, datasets, and reusable machine learning experiments.

The goal is not only to improve leaderboard scores, but also to build a public record of how I think, experiment, fail, improve, and communicate technical insights.

## Why this repository exists

I am using Kaggle as a practical machine learning lab.

Each competition or dataset project will include:

- problem understanding
- baseline model
- exploratory data analysis
- validation strategy
- experiment logs
- reusable Python utilities
- reflections for developers
- LinkedIn post drafts

```text

```

## My learning principles

1. Start with a baseline.
2. Trust local validation before the public leaderboard.
3. Log every meaningful experiment.
4. Refactor repeated code into reusable modules.
5. Share insights, not just results.
6. Make progress visible.

## Getting started

Create a virtual environment and install the starter dependencies:

```bash
python -m venv .venv
pip install -r requirements.txt
```

When starting a new competition, copy `competitions/_template` and rename the copy to the Kaggle competition slug.
