# Titanic Submission Checklist

## Before Running

- [ ] Join competition
- [ ] Open Data tab
- [ ] Confirm `train.csv`, `test.csv`, and `gender_submission.csv` are visible
- [ ] Create or copy a Kaggle notebook

## Before Submitting

- [ ] `submission.csv` exists
- [ ] Columns are exactly `PassengerId,Survived`
- [ ] Row count matches `test.csv`
- [ ] No missing values
- [ ] `Survived` contains only `0` and `1`

## After Submitting

- [ ] Record public score
- [ ] Save screenshot
- [ ] Update `experiment_log.md`
- [ ] Commit result notes to GitHub
