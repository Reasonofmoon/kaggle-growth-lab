# Competitions

This folder contains Kaggle competition workspaces.

Use `competitions/_template` as the starting point for each new competition:

```text
competitions/[competition-slug]/
├── README.md
├── notebooks/
├── src/
├── submissions/
└── experiment_log.md
```

Keep raw Kaggle data outside Git. Document how to download it instead of committing files from `data/`, `input/`, or downloaded archives.
