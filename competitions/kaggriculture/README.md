# Competition: Kaggriculture Capstone

Kaggriculture is the capstone challenge for the 5-Day AI Agents: Intensive Vibe Coding Course with Google.

The course description frames it as a farming simulation where an autonomous agent manages resources, responds to a changing in-game economy, competes for limited resources, and tries to outperform other agents.

## 1. Problem

Build and deploy an autonomous AI agent for a virtual farming simulation.

Detailed requirements are expected to be released during the course week.
