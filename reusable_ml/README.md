# Reusable ML

This folder contains small utilities that can be reused across Kaggle competitions.

The goal is to extract repeated workflow code from notebooks into testable Python modules.

Starter modules:

- `validation.py`: cross-validation split helpers
- `metrics.py`: metric helpers and wrappers
- `visualization.py`: common chart helpers
- `feature_tools.py`: feature engineering helpers
- `experiment_tracker.py`: lightweight experiment logging
